Stargazing App

The stargazing app is a cross platform application that aims to share the stories and legends that have been told about the stars. It was built using Unity, and allows the user to look around the night sky using their phone or tablet and learn about constellations from the perspectives of many cultures.

Snow Accumulation Shader

For my term project in Graphics 2, I created a surface shader for Unity to simulate snow accumulation. Snow in video games is usually accomplished by either using a pre-modelled snow mesh, or by adding a flat snow shader to existing models, resulting in snow that cannot be affected by changes in weather. By allowing the depth and accumulation direction to vary depending on the weather, I created snow that is more dynamic and interesting than static meshes.
